## Prerequisites

All tutorials assume you have:

- **Python 3.11+** installed
- **Git** installed
- A **GitHub account**
- A terminal / command line

The beginner tutorial covers cloning and setup from scratch. Intermediate and advanced tutorials assume you've completed the beginner tutorial or the [Hackathon Setup](../hackathon/setup.md).
