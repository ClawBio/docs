## Prerequisites

All tutorials assume you have:

- **Python 3.11+** installed
- **Git** installed
- A **GitHub account**
- A terminal / command line

The beginner tutorials cover setup from scratch. Intermediate and advanced tutorials assume you've completed the setup tutorial or the [Hackathon Setup](../hackathon/setup.md).
