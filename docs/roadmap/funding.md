# Funding

ClawBio's funding strategy starts with non-dilutive grants in 2026 and transitions to sustainable revenue from Q4 onwards.

## Non-Dilutive Funding (2026)

| Source | Type | Notes |
|--------|------|-------|
| Wellcome Trust | Open Research Fund / Discovery Award | Reproducibility and open science alignment |
| UKRI Future Leaders Fellowship | Fellowship | Infrastructure for bioinformatics standards |
| Innovate UK Smart Grants | Innovation grant | Platform development and clinical readiness |
| Hackathon prizes | Prize funding | Bioinformatics and AI agent competitions |

Grant applications will emphasise ClawBio's contributions to:

- **Reproducibility**: SHA-256-verified skill bundles replacing ad hoc scripts
- **Open science**: DOI-minted, openly licensed research artefacts
- **Health equity**: HEIM-scored datasets and equity-aware skill design
- **Clinical safety**: governed validation tiers for pharmacogenomics and clinical skills

## Revenue Model (Q4 2026+)

### ClawBio Pro

Institutional subscription for private, governed skill management.

| Feature | Included |
|---------|----------|
| Private skill hubs | Isolated registry for internal skills |
| RBAC | Role-based access control (admin, author, viewer) |
| SSO | SAML and OIDC integration |
| Audit logging | Full execution and access trail |
| Priority support | Dedicated support channel |
| **Pricing** | **GBP 500 / month / institution** |

### Hosted Execution

- Pay-per-run pricing for skills executed via the HTTP API
- Sandboxed, authenticated, rate-limited
- Ideal for web applications and third-party integrations

### Enterprise Support

- Custom skill development and validation
- On-site training and onboarding
- SLA-backed support

### Training Workshops

- Instructor-led workshops at conferences and institutions
- Topics: skill authoring, SKILL.md specification, agent integration, clinical validation
- Revenue from registration fees

## Sustainability

The open-source core (SKILL.md spec, CLI tools, community skills) remains free and openly licensed. Revenue from ClawBio Pro and services funds continued development, community programmes, and infrastructure costs.
