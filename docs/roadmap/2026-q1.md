# Q1: Specification (Jan -- Mar 2026)

The first quarter focuses on solidifying the SKILL.md specification, publishing the first verified skills, and submitting the initial academic communication.

## Key Deliverables

### SKILL.md v2.0 Specification

- Full specification published with formal JSON schema
- Complete PharmGx example skill demonstrating every field and section
- RFC process established for schema evolution (30-day public comment period)
- Versioned specification hosted on clawbio.ai

### Verified Skill Catalogue

- **20 verified skills** across 5 domains:
  - Pharmacogenomics (PharmGx Reporter, ClinPGx, NutriGx Advisor)
  - Population genetics (Equity Scorer, Genome Compare, Ancestry PCA)
  - Single-cell biology (scRNA Orchestrator, scRNA Embedding)
  - Variant annotation (GWAS PRS, GWAS Lookup, VCF Annotator)
  - Infrastructure (Bio Orchestrator, UKB Navigator, Galaxy Bridge)
- Every skill includes demo data, CLI invocation, and passing tests

### Academic Output

- **Nature Biotechnology Correspondence** submitted introducing the SKILL.md standard
- Preprint deposited on bioRxiv for community feedback

### Contributor Onboarding

- Contributor onboarding workflow documented in `CONTRIBUTING.md`
- Skill template (`SKILL-TEMPLATE.md`) refined with community feedback
- First external contributor PRs merged
