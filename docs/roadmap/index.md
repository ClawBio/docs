# Strategic Roadmap 2026

ClawBio's roadmap is organised into four quarters, each building on the last. Every milestone is tied to an adoption metric so progress is measurable.

## Success Metrics

| Metric | Target |
|--------|--------|
| Skills on hub | 200 verified |
| Monthly installs | 1,000+ |
| Unique skill authors | 50+ |
| Pilot institutions | 20 groups |
| Publications citing ClawBio | 10+ |
| GitHub stars | 1,000 |
| Benchmarks published | 2 preprints |
| DOI-minted skills | 50+ |
| MCP-bridged skills | 100+ |
| Time-to-skill (median) | < 1 hour |
| Time-to-integration | < 30 min |

## Quarterly Overview

- **[Q1: Specification](/roadmap/2026-q1)** — Jan -- Mar 2026. SKILL.md v2.0, 20 verified skills, Nature Biotechnology Correspondence.
- **[Q2: Platform](/roadmap/2026-q2)** — Apr -- Jun 2026. clawbio.ai hub, authoring toolkit, MCP bridge, DOI minting.
- **[Q3: Community + Evidence](/roadmap/2026-q3)** — Jul -- Sep 2026. 100 skills, reproducibility benchmarks, pilot groups, workshops.
- **[Q4: Scale + Clinical Readiness](/roadmap/2026-q4)** — Oct -- Dec 2026. 200+ skills, ClawBio Pro, clinical-grade validation, platform paper.
